# E-Commerce Sales Dashboard using Tableau

## Project Overview
This project presents an interactive E-Commerce Sales Dashboard built using Tableau. The dashboard is designed to analyze sales performance, customer behavior, product trends, revenue growth, and business KPIs through interactive data visualizations and business intelligence reporting.

The project transforms raw e-commerce data into meaningful business insights using Tableau dashboards, filters, charts, and analytical visualizations.

---

# Objectives
- Analyze overall sales and revenue performance
- Track category-wise and monthly sales trends
- Identify top-performing products and customer segments
- Understand customer purchasing behavior
- Monitor profitability and return trends
- Create interactive business intelligence dashboards

# Business Insights
- Electronics category generated the highest revenue.
- UPI was the most preferred payment method among customers.
- Premium customers contributed significantly to total revenue.
- Certain states generated higher order volumes and profitability.
- Monthly sales trends revealed seasonal buying patterns.
- Return requests were concentrated in selected product categories.
